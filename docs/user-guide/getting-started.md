# Getting Started with MemoryOS

## Step 1: Install

Download the latest release for your platform from [GitHub Releases](https://github.com/your-org/memoryos/releases).

| Platform | File |
|----------|------|
| Windows | `memoryos-vX.X.X-windows.zip` |
| macOS | `memoryos-vX.X.X-macos.dmg` |
| Linux | `memoryos-vX.X.X-linux.tar.gz` |
| Android | `memoryos-vX.X.X-android.apk` |

## Step 2: First Launch

On first launch, MemoryOS will guide you through onboarding:
1. Choose watched directories
2. Download an AI model (optional, recommended: Qwen 2.5 1.5B)
3. Start indexing

## Step 3: Add Files

**Auto-indexing**: Add folders in Settings → Watch Directories. MemoryOS monitors them automatically.

**Manual import**: Click **+ Add Files** on the home screen.

## Step 4: Search

Type any natural language query in the search bar:
- "Find my AWS security notes"
- "Show invoices from 2023"
- "Kubernetes deployment guides"

## Step 5: AI Chat

Navigate to **AI Chat** and ask questions like:
- "What did I learn about Docker?"
- "Summarize my cloud knowledge"

!!! note
    AI Chat requires a downloaded model. Go to **Settings → AI Models** to download one.

## Step 6: Secure Vault

Add sensitive files to the **Vault** for encrypted storage. Unlock with biometrics or a password.
